Cheat sheet creating

You can create your own cheat sheet using our website. All cheat sheet are publicly available, so creating one yourself means you are ready to contribute to the website, helping out the users too! Markdown and LaTeX are supported!
The only requirement is to get an account.

Available in one simple click!

Cheat sheet critic

All cheat sheet have a comment section. Let's privilege constructive criticism in order for the author to edit what is incorrect.
This requires to get an account too.
